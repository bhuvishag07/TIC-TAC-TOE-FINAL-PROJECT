🎮 TTT Arena – Interactive Tic Tac Toe Web Platform

TTT Arena is a modern browser-based Tic Tac Toe game that enhances the traditional gameplay experience with additional interactive features such as leaderboards, badge rankings, theme customization, and a community forum.

Unlike basic Tic Tac Toe implementations, this project focuses on user engagement and interactive design, transforming a simple game into a mini web gaming platform.

The application is built entirely using HTML, CSS, and JavaScript, and runs directly in the browser without requiring any installation or backend server.

🧠 How the Game Works

The game uses a 9-cell board array representing the Tic Tac Toe grid.

After each move, the program checks predefined winning combinations:

Rows

Columns

Diagonals


If three matching symbols (X or O) appear in a winning pattern, the game declares the winner and highlights the winning cells.

If all cells are filled without a winner, the game ends in a draw.
